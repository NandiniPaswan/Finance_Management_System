# Finance_Management_System

## Description
A web-based application to track income, expenses,bills and generate reports.

## Features
- Add income & expenses
- Add bills
- Monthly Income vs Expense Bar Chart visualization
- Category-wise spending Pie Chart analysis
- Track bills with Paid vs Pending Pie Chart
- User-friendly dashboard

## Technologies
- Frontend: HTML,CSS,Javascript
- Backend: Python (Flask)
- Database: SQLite

##How to Run the Project
1. Install Python
2. Install required libraries:
   -pip install flask
3. Run the backend:
   -python api.py
4. Open index.html in your browser
